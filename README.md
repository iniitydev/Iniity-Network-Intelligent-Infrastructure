### The `IniityEvent`: The Atomic Unit of Verifiable Truth

At the heart of the fabric is the `IniityEvent`. This is the canonical data structure that makes cognition verifiable. It is not a log entry; it is a signed, structured, and context-rich record containing the **WHO, WHAT, WHEN, WHERE, WHY, and HOW** of every occurrence.

-   **WHO (Identity):** Which sovereign `AuthO.iD` entity initiated or was affected by the event?
-   **WHERE (Space):** At what physical and digital coordinates did the event occur?
-   **WHY (Purpose):** Which strategic goal or policy was this event intended to serve?
-   **HOW (Planning):** Which orchestrated workflow generated this event?

This structure ensures that every action is fully contextualized, auditable, and traceable back to a sovereign identity and a defined purpose.
